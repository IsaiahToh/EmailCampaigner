How to Use

  1. Upload an Excel file containing email addresses (one per row in the first column)
  2. Upload your promotional flyer as an image or create one using HTML
  3. Configure your email settings (sender info, subject, etc.)
  4. Click "Send Emails" to start your campaign
  5. Monitor progress in real-time

Important Note

This demo uses EmailJS for sending emails. To make it work, you'll need to:

  1. Sign up at EmailJS.com
  2. Create a service (like Gmail or SendGrid)
  3. Create an email template with these variables: {{from_name}}, {{to_email}}, {{subject}}, {{{flyer_content}}}
  4. Use triple braces {{{flyer_content}}} for HTML content to render properly
  5. Set environment variables in Render deployment

📧 EmailJS Template Example:

Subject: {{subject}}

Dear Customer,

{{{flyer_content}}}

Best regards,
{{from_name}}
